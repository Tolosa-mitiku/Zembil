STEP 1: Open MongoDB Compass
-----------------------------
1. Launch MongoDB Compass
2. Connect to your MongoDB instance
3. Select or create your database (e.g., "zembil")


STEP 2: Import Collections (One by One)
----------------------------------------
For each JSON file in this folder, follow these steps:

1. In Compass, click on your database name
2. Click "CREATE COLLECTION" or select existing collection
3. Name it exactly as the file name (without .json):

   - buyers
   - sellers
   - categories
   - products
   - orders
